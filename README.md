# 一款基于标签的个人知识管理系统

![screenshot](http://farm5.static.flickr.com/4111/5171416767_290c6a21c7.jpg)

## 环境要求

- php5+
- apache/nginx/...

## 使用说明

### 知识源文件

将文件解压后，在文件夹的根目录，新建data.txt文件，这个文件就是知识库，所以一定要记得备份该文件。

### data.txt的书写规范

	{{{电影 公路 北野武 日本 温情 成长 久石让

#### 注意点
- 文章前后用{{{和}}}作为分隔符
- {{{后面是标签，用空格分开
- 第一行一定是 "# 标题" 这种格式

### 浏览

打开浏览器，访问该文件夹的web路径，如"http://localhost/pkm"
